# Everything Football

## This space will be used to analyze football :soccer: through a number of different datasets. 

### Datasets:
- [Fivethirtyeight](https://github.com/fivethirtyeight/data/tree/master/soccer-spi)
    - `spi_matches` contains match-by-match SPI ratings and forecasts back to 2016
    - `spi_global_rankings` contains current SPI ratings and rankings for men's club teams
    - `spi_global_rankings_intl` contains current SPI ratings and rankings for men's international teams
